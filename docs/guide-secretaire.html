Votre rôle

En tant que secrétaire de mairie, vous êtes responsable de la mise à jour du contenu du site communal : les pages d'information, les documents officiels, et la galerie de photos.

Concrètement, vous pouvez :

  • Modifier les pages du site (textes, images, mise en page)
  • Publier ou dépublier des pages
  • Ajouter des documents (PV de conseil, bulletins, arrêtés, PLU...)
  • Gérer la galerie de photos de la commune
Bon à savoir : Vous n'avez pas accès au module participatif (sollicitations des habitants). Ce module est réservé aux élus et au maire. Votre domaine est le contenu éditorial du site.

Accéder à l'espace d'administration

La gestion du contenu se fait via l'interface d'administration de Django, un espace séparé du site public.

Depuis le menu du site

  1. Connectez-vous sur le site avec votre compte secrétaire
  2. Ouvrez le menu « Saulzet & Vous » dans la barre de navigation
  3. Cliquez sur « Édition » — vous arrivez directement dans l'espace de gestion des pages et documents

Par l'adresse directe

  1. Rendez-vous à l'adresse saulzet-le-froid.com/admin/
  2. Connectez-vous avec votre email et votre mot de passe
  3. Vous arrivez sur le tableau de bord d'administration
Note : L'accès à l'administration nécessite un mot de passe. Le lien magique ne fonctionne pas pour l'administration. Si vous avez oublié votre mot de passe, contactez l'administrateur du site.

Dans l'espace d'administration, vous verrez la section « Pages » qui contient tout ce dont vous avez besoin.

Gérer les pages du site

Voir la liste des pages

  1. Dans l'administration, cliquez sur « Pages »
  2. Vous voyez la liste de toutes les pages du site avec leur titre, statut (publiée ou non), et date de mise à jour

Modifier une page existante

  1. Cliquez sur le titre de la page que vous souhaitez modifier
  2. L'éditeur s'ouvre avec le contenu actuel de la page
  3. Effectuez vos modifications (voir la section « Éditeur de texte » ci-dessous)
  4. Cliquez sur « Enregistrer » en bas de la page

Les champs d'une page

ChampÀ quoi il sert
Titre Le titre affiché en haut de la page et dans le menu
Contenu Le texte principal de la page (avec l'éditeur visuel)
Extrait Un court résumé (facultatif, utilisé dans les listings)
Page parente Si la page est une sous-page (ex : « Équipe » est enfant de « Mairie »)
Publiée Cochez pour rendre la page visible sur le site. Décochez pour la masquer temporairement.
Afficher dans le menu Cochez pour que la page apparaisse dans la barre de navigation du site
Ordre dans le menu Un nombre pour définir la position dans le menu (10, 20, 30... le plus petit en premier)
Modèle Le type de mise en page (voir « Pages spéciales »)
Description SEO Courte description pour les moteurs de recherche (facultatif, 160 caractères max)

Publier ou dépublier une page

Pour masquer temporairement une page (par exemple pendant une mise à jour) : décochez la case « Publiée » et enregistrez. La page ne sera plus visible sur le site mais son contenu est conservé.

Attention : Dépublier une page la rend invisible à tous les visiteurs. Vérifiez que ce n'est pas une page importante avant de la dépublier.

Utiliser l'éditeur de texte

L'éditeur de texte (TinyMCE) fonctionne comme un traitement de texte. Vous n'avez pas besoin de connaître le code HTML.

Les outils principaux

BoutonAction
BMettre le texte en gras
IMettre le texte en italique
Liste à pucesCréer une liste avec des puces
Liste numérotéeCréer une liste numérotée (1, 2, 3...)
LienInsérer un lien vers une autre page ou un site web
ImageInsérer une image dans le texte
TableauInsérer un tableau (lignes et colonnes)
FormatChanger le style du texte (titre, sous-titre, paragraphe)

Insérer un lien

  1. Sélectionnez le texte qui doit devenir un lien
  2. Cliquez sur le bouton « Lien » (icône de chaîne)
  3. Entrez l'adresse (URL) :
    • Pour un lien vers une page du site : /mairie/equipe/
    • Pour un document : /media/documents/nom-du-fichier.pdf
    • Pour un site externe : https://www.example.com
  4. Cliquez sur « OK »

Insérer une image

  1. Cliquez sur le bouton « Image »
  2. Entrez le chemin de l'image : /media/pages/images/nom-de-limage.jpg
  3. Ajoutez une description (texte alternatif, pour l'accessibilité)
  4. Cliquez sur « OK »
Les images doivent d'abord être envoyées sur le serveur dans le dossier media/pages/images/. Demandez à l'administrateur si vous ne savez pas comment faire.

Gérer les documents officiels

Les documents (procès-verbaux, bulletins, arrêtés, PLU...) sont gérés depuis l'administration et apparaissent sur la page « Documents » du site.

Ajouter un document

  1. Dans l'administration, ouvrez la page « Documents » (ou la page à laquelle rattacher le document)
  2. En bas de la page, vous trouverez la section « Documents »
  3. Cliquez sur « Ajouter un document »
  4. Remplissez les champs :
    • Titre : le nom du document (ex : « PV du conseil du 15 mars 2026 »)
    • Fichier : cliquez sur « Choisir un fichier » pour sélectionner le PDF sur votre ordinateur
    • Catégorie : sélectionnez le type de document
    • Date : la date du document (pas la date de mise en ligne)
  5. Cliquez sur « Enregistrer »

Catégories de documents

Les catégories servent à classer les documents et permettent aux visiteurs de filtrer la liste. Par défaut, cinq catégories sont préconfigurées : Procès-verbaux, Bulletins, PLU, Arrêtés et Autre.

Les catégories sont paramétrables : vous pouvez en ajouter, les renommer ou les réorganiser directement depuis l'administration.

Ajouter ou modifier une catégorie

  1. Dans l'administration, cliquez sur « Catégories de documents »
  2. Pour ajouter : cliquez sur « Ajouter », remplissez le nom et enregistrez (le slug se génère automatiquement)
  3. Pour modifier : cliquez sur le nom de la catégorie et changez-le
  4. Utilisez le champ « Ordre » pour définir l'ordre d'affichage des filtres sur le site

Supprimer un document

Dans l'administration, ouvrez la page contenant le document, cochez la case à côté du document à supprimer, puis enregistrez. Le fichier sera retiré du site.

Gérer la galerie de photos

La galerie de photos est une page spéciale qui affiche les images de la commune avec un effet de visionneuse (lightbox).

Ajouter des photos

  1. Dans l'administration, ouvrez la page qui utilise le modèle « Galerie »
  2. En bas, vous trouverez la section « Photos de la galerie »
  3. Cliquez sur « Ajouter une photo »
  4. Remplissez :
    • Image : sélectionnez la photo sur votre ordinateur
    • Titre (légende) : un court texte décrivant la photo
    • Crédit : le nom du photographe (facultatif)
    • Ordre : un nombre pour définir la position dans la galerie
    • Publiée : cochez pour rendre la photo visible
  5. Cliquez sur « Enregistrer »
Bon à savoir : Les photos sont automatiquement redimensionnées (maximum 1920 pixels de large) et une miniature est créée automatiquement. Vous n'avez pas besoin de les réduire avant de les envoyer.

Réorganiser la galerie

Changez le champ « Ordre » de chaque photo pour les réorganiser. Utilisez des nombres espacés (10, 20, 30...) pour pouvoir insérer facilement de nouvelles photos entre deux existantes.

Pages spéciales

Certaines pages du site utilisent des modèles spéciaux qui ajoutent des fonctionnalités automatiques en plus du contenu que vous éditez.

ModèleFonctionnalité ajoutéeExemple de page
Par défaut Aucune — affiche simplement le contenu La plupart des pages
Contact Ajoute un formulaire de contact en dessous du contenu Page « Contact »
Documents Affiche la liste des documents classés par catégorie Page « Documents »
Équipe Affiche automatiquement les photos et fonctions des élus Page « L'équipe municipale »
Galerie Affiche la galerie de photos avec visionneuse Page « Galerie photos »
Habitants Affiche un graphique démographique Page « La commune en chiffres »
Accès Affiche une carte interactive avec les villages Page « Plan d'accès »
Ne changez pas le modèle d'une page spéciale sans en parler à l'administrateur. Changer le modèle peut désactiver des fonctionnalités (formulaire de contact, carte...).

Bonnes pratiques

Enregistrez régulièrement

Cliquez sur « Enregistrer » fréquemment pendant l'édition pour éviter de perdre votre travail. L'éditeur ne sauvegarde pas automatiquement.

Nommez bien vos documents

Utilisez des noms de fichiers clairs et datés, par exemple : pv-conseil-2026-03-15.pdf plutôt que scan001.pdf. Les habitants les retrouveront plus facilement.

Vérifiez sur le site

Après chaque modification, ouvrez la page concernée sur le site public pour vérifier que le résultat correspond à ce que vous attendez.

Ne supprimez pas les pages existantes

Si une page n'est plus utile temporairement, dépubliez-la (décochez « Publiée ») plutôt que de la supprimer. Cela conserve le contenu au cas où.

Taille des images

Privilégiez des images au format JPG pour les photos et PNG pour les logos ou images avec fond transparent. Évitez les images de plus de 5 Mo.
